# Web-repository

## Tarea hecha en clases

### Objetivo:

Corregir el uso de css en HTML (Inline)

### Solucion:

Algunos textos requerrian cambio de colores y
algunas imagenes necesitaban cambio de tamaño de imagen, 
estaban ingresados las configuraciones en el HTML desde dentro (Inline). La solucion fue crear clases de algunos textos e ingresarlos en un CSS ya previo que tenia estos archivos para tenerlos de manera correspondiente y que no sea visible para el
usuario que quiera ver la pagina web.
